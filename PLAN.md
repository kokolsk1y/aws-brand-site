### Фаза 2: AI-контент (генерация изображений)

**Всего нужно: ~20-25 изображений + 2-3 видео**

#### 2.1 Hero-слайдер (6 фото)
Полноэкранные интерьерные фото с розетками/выключателями на стене.
Стиль: скандинавский минимализм, мягкий свет, как editorial photography.
```
Промпт-шаблон для Midjourney:
"modern scandinavian interior, [тип комнаты], white/light walls, 
elegant light switch on wall, soft natural lighting, 
editorial architectural photography, 4K, shallow depth of field"

Варианты комнат: гостиная, спальня, кухня, ванная, прихожая, офис
```

#### 2.2 Коллекции Aura и Uno (4 фото)
Крупный план: розетка/выключатель в реальном интерьере, фокус на продукте.
```
Промпт:
"close-up of modern minimalist white light switch on textured wall, 
[описание стены], interior design photography, soft shadows, 
macro detail shot, premium product photography"
```

#### 2.3 Lifestyle (3 фото)
Рука нажимает выключатель, тёплая атмосфера.
```
Промпт:
"hand gently pressing modern white light switch, warm interior lighting, 
close-up, shallow depth of field, lifestyle photography, cozy atmosphere"
```

#### 2.4 Полноэкранные панорамы (3 фото)
Для parallax-фонов, без фокуса на продукте — чистые интерьеры.
```
Промпт:
"wide angle modern interior panorama, minimalist design, 
natural light through large windows, soft tones, 
architectural photography, 16:9 aspect ratio"
```

#### 2.5 Продукт на абстрактном фоне (4 фото)
Для горизонтального скролла коллекций.
```
Промпт:
"single modern light switch floating on abstract soft green gradient background,
studio lighting, product photography, clean minimal, centered composition"
```

#### 2.6 Видео (Runway, 2-3 шт)
Берём лучшие фото из п.2.1 и 2.3, делаем 4-10 сек видео:
- Камера медленно приближается к выключателю
- Рука плавно нажимает кнопку
- Свет включается, комната освещается
